# NgxToggles

This sample project experiments dynamic load in Angular 2+ framework.

## Build & run

```
npm i
npm start
```
Navigate to `http://localhost:4200/`. 
The app will automatically reload if you change any of the source files.

## Experiments

### step1: load a component

git checkout step1
